    Имя Тип Адрес / URL В режиме монитора Действия
    VM/CT для мониторинга

    Введите ID или имя VM/CT и нажмите «Добавить» — гость будет отображаться в режиме монитора. Список загружается с Proxmox по кнопке «Обновить».

    Имя Тип Статус Примечание В режиме монитора Действия
    Импорт / экспорт
    Только хосты мониторинга

    Экспорт или импорт только списка хостов для мониторинга сервисов.

    Только VM/CT

    Экспорт или импорт только списков мониторинга VM/CT и скрытых на экране монитора.

    Полная конфигурация

    Экспорт или импорт всех настроек, подключений (включая токены) и списка хостов мониторинга. Используйте осторожно.
