Порядок при перелистывании (свайп, стрелки на панели). Экран бэкапов доступен только при подключении к Proxmox.

    Стили
    Настройки меняют оверрайды для карточек, заголовков, таблиц и прогресса. Оверрайды применяются только к выбранному варианту.
    Пороги индикации
    CPU: Зелёный до 70%
    CPU: Жёлтый до 90%
    RAM: Зелёный до 70%
    RAM: Жёлтый до 90%
    Безопасность

    Вы можете защитить доступ к настройкам паролем.

    Повторный ввод пароля не требуется в течение этого времени после входа.
    Хосты для мониторинга

    Добавьте хосты или URL для проверки доступности. В разделе «Мониторинг сервисов» отображается только список и статус.

    Имя Тип Адрес / URL Иконка В режиме монитора Действия
    UPS (NUT/SNMP)
    Сетевые устройства (SNMP)
    Поля (до 15)
    Добавляйте и удаляйте поля кнопками; переключатель «Опрашивать» отключает SNMP по строке без удаления. Пустой OID на дашборде не показывается. «Время» — SNMP TimeTicks; «МБ/ГБ» — значение в байтах, единицы из настроек «Единицы измерения». «Статус (bool, 0/1 — авто)» и «Статус (вручную)» — как в подсказках под списками значений.
    Метрики хостов Proxmox

    Для каждого узла можно выбрать endpoint агента, датчик температуры CPU и сетевой интерфейс для скорости линка.

    Backend не будет опрашивать источник чаще этого интервала.
    Сколько ждать ответ от endpoint узла.
    Свежие данные будут отдаваться из кэша в пределах этого TTL.
    Если температура достигнет порога, рядом с узлом появится предупреждение.
    Если линк будет ниже порога или перейдет в down, рядом с узлом появится предупреждение.
