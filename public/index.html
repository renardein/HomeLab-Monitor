    Speedtest

    Требуется официальный CLI Ookla Speedtest в PATH на сервере (команда speedtest).

    ID сервера Ookla (число). Пусто — авто.

    Необязательно: для сравнения с замерами и шкалы прогресс-баров скачивания. Пусто — шкала по фактическим значениям за сегодня.

    Прокси для CLI

    Передаются в процесс Ookla Speedtest как переменные окружения. Для HTTPS-API часто нужен отдельный HTTPS_PROXY. Пустые поля — без прокси (кроме системных переменных среды Node).
