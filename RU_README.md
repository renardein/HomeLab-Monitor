# HomeLab Monitor

[English](./README.md) | [Русский](./RU_README.md)

Локальная веб-панель для Proxmox и TrueNAS с отдельными макетами обычного и monitor-режима.

## Возможности

- мониторинг кластера Proxmox VE
- мониторинг TrueNAS CORE / SCALE
- проверка сервисов по TCP, UDP, HTTP(S), SNMP и NUT
- карточки VM / CT с кастомными иконками и цветами
- мониторинг UPS и сетевых SNMP-устройств
- отдельный экран Speedtest
- дополнительные метрики хостов Proxmox: температура CPU и скорость линка с помощью [Node.js агента](./docs/proxmox_agent_manual_ru.md)
- полноэкранный режим монитора с навигацией стрелками и свайпами
- по умолчанию monitor mode использует светлую тему
- пароль на настройки, импорт / экспорт и локальное хранение в SQLite
- **мастер первого запуска**: язык, выбор Proxmox или TrueNAS, подключение либо **импорт полного JSON-бэкапа** (тот же формат, что и «Настройки» → экспорт) для восстановления конфигурации за один шаг
- **интеграция Telegram**: токен бота, правила по типам событий (сервисы, VM/CT, узлы, SNMP-слоты, температура CPU на узле, скорость линка), при необходимости **свои шаблоны сообщений** с подстановками по типу правила (редактирование в отдельном окне)
- **агент host metrics**: установка и снятие агента на узле Proxmox **через SSH из интерфейса**, удаление агента с узла, когда он больше не нужен

## Быстрый запуск

Требования:

- Node.js 16+

Установка зависимостей:

```bash
npm install
```

Запуск приложения:

```bash
npm start
```

Режим разработки:

```bash
npm run dev
```

Агент host metrics:

```bash
npm run host-metrics-agent
```

Адрес по умолчанию:

```text
http://localhost:81
```

## Настройка

1. Открой приложение в браузере. При первом запуске (или после полного сброса настроек) пройди **мастер начальной настройки** или импортируй JSON-бэкап.
2. Если не сделал это в мастере, в `Settings` добавь подключение к Proxmox и/или TrueNAS.
3. Настрой сервисы, VM / CT, UPS, SNMP-устройства, Speedtest и host metrics.
4. При необходимости измени порядок экранов monitor mode, темы, параметры отображения и видимость экранов.
5. При необходимости настрой **Telegram** в разделе настроек.
6. При необходимости включи пароль на настройки и используй импорт / экспорт.
